MultiHeadAttention

class MultiHeadAttention(val heads: List<ScaledDotProductAttention>, val queryProjections: List<Tensor>, val keyProjections: List<Tensor>, val valueProjections: List<Tensor>) : Attention

MHA and MMHA are composed of the following operations:

  • Linear value input (|V| = d_v)

  • Linear key input (|K| = d_k)

  • Linear query input (|Q| = d_k)

  • Scaled Dot Product Attention x Head Count (h)

  • Concat

  • Linear
