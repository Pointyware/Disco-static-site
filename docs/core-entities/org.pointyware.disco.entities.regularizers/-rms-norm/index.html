RmsNorm

class RmsNorm(val count: Int) : Regularizer

RmsNorm is typically used within a layer to normalize the activations from a previous layer or sum of activations before calculating the preactivation weighted sum.

Normalizers can also be seen as dedicated layers with an activation function that acts on the whole instead of the parts, without weights or biases.
